# Project Requirements
A data pipeline built with Python using pandas and sql, using ETL to load a dataset, clean/tranform it, and store output into a data warehouse.

## Software Requirements
- Python 3+
- Pandas
- SQL (SQLite/Postgres/other)
- Data Warehouse
- Large dataset
- Logging (software/data lineage)
- Classes and modules
- Hosted on GitHub

## Optional
- Numpy
- Matplotlib (or any other visualization)
- AWS (RDS or EC2)
- Advanced data formats (Parquet)
- Daemon process (ideally with cron)
